# Eval Rubric

Use this rubric after implementation. Score each category `0-2`.

## Categories

1. Hierarchy clarity
   - 0: unclear structure, weak scanning
   - 1: mostly clear with minor confusion
   - 2: clear priority and section flow
2. Anti-template quality
   - 0: obvious AI-template feel
   - 1: partially differentiated
   - 2: distinct and intentional visual language
3. Typography quality
   - 0: generic/default typography choices
   - 1: some tuning but inconsistent
   - 2: coherent weights, tracking, and reading measure
4. Component polish
   - 0: muddy borders/shadows, clunky controls
   - 1: acceptable polish with minor flaws
   - 2: crisp controls and consistent component system
5. Asset integrity
   - 0: missing/broken/hidden assets
   - 1: all assets show but with minor issues
   - 2: all assets correct and purposeful

## Ship Bar

- Target: `>= 8/10`.
- Mandatory pass condition: asset integrity must be `2/2` when assets are part of scope.
- If "two images not showing" exists, result is auto-fail until fixed.
