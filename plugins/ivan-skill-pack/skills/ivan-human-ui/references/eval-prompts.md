# Eval Prompts

Use these prompts to evaluate whether the skill generalizes beyond one demo page.

## Prompt 1: Marketing Hero Cleanup

"Refactor this SaaS marketing homepage to remove AI-template vibe. Keep the same content, but improve hierarchy, typography, and CTA polish. No framework migration."

Pass criteria:

- Clear before/after structure change
- No default purple/indigo AI palette drift
- Concrete value-level edits, not vague advice

## Prompt 2: Dashboard Landing Polish

"Use real product screenshots and redesign this dashboard landing page so it feels human-crafted and calm. Keep neutral color direction. Improve section rhythm."

Pass criteria:

- Screenshot treatment looks intentional
- Ring/shadow/border layering is crisp
- Spacing is numerically consistent

## Prompt 3: Broken Testimonials Media

"Two testimonial backgrounds are not showing. Fix all media visibility issues and finish the anti-AI visual polish."

Pass criteria:

- All required tracks render: animal, abstract, business
- Paths/extensions/casing and clipping issues resolved
- Final output includes explicit verification checklist

## Prompt 4: Existing Page Enhancement

"Do not rewrite from scratch. Improve this existing page to remove AI flavor while preserving current structure and copy."

Pass criteria:

- Keeps existing structure intent
- Raises craft quality without destructive rewrites
- Eval rubric score reaches ship bar
