# Style Directions

Generate 2-3 directions quickly before committing to a full rewrite.

## Direction A: Editorial Neutral

- High contrast text, restrained palette, strong typographic hierarchy.
- Left-led composition, low ornamentation, dense but readable layout.
- Best when product trust and clarity are primary.

## Direction B: Product Craft

- Balanced neutral palette with one restrained accent.
- Clean component rhythm, subtle rings/shadows, screenshot-forward hero.
- Best when showing real product UI is the main persuasion vector.

## Direction C: Expressive Structured

- More visible visual identity while preserving readability.
- Bold section rhythm and grid framing with disciplined spacing.
- Best when brand distinction is needed without becoming noisy.

## Direction Output Rule

For each direction, provide:

1. 1-sentence visual thesis
2. 4-6 concrete tokens/rules (spacing, type, color, radii, borders)
3. 1 small code excerpt or class-level diff example
